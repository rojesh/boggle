# README FOR SETUP

## Clone the repo
```
git clone https://github.com/rojesh/boggle
```

## Go to the app directory
```
cd boggle
```

## Setup Backend
```
bundle install  
bundle exec rspec
```

## Setup Frontend
```
cd boggle-app  
npm install
```

## Run in local environment
```
cd ..
heroku local -f Procfile.dev
```
